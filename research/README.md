# Skill Research Briefs

This folder contains research briefs for future commerce-agent skills.

## What Belongs Here

- Role definitions and workflow notes
- API and MCP research
- Scope boundaries
- Output ideas and trigger phrases

## What Does Not Belong Here

- Installable skill packages
- `SKILL.md` entrypoints intended for direct agent use
- Packaged assets, evals, or validation scripts unless the brief is being promoted into a real skill

## How To Use This Folder

- Use the repo-root `skills/` folder for packaged, runnable skills.
- Use `research/skills/` for idea-stage role briefs and implementation planning.
- When a research brief is mature enough, promote it into a packaged skill under the repo-root `skills/` folder.
